# instadownloader is a script for taking photos/videos of someone on Instagram by entering the post link

   #How To Install In Linux&Termux

    Manual Method
    ======================================
    $apt update
    $apt upgrade
    $apt install python
    cd instadownloader
    pip install instaloader
    python3 ID.py


    Auto Method
    ======================================
    $apt update
    $apt upgrade
    $apt install bash
    cd instadownloader
    bash run.sh
